# Glosario de comandos de Bash

~~~
mkdir: Este comando sirve para crear un nuevo directorio

cd: Sirve para movernos entre directorios o tambien para poder retroceder en el caso de poner 2 puntos. 

ls: Nos muestra los archivos existentes en el directorio actual.

echo: Este comando sirve para insertar texto en un archivo.

help: Nos muestra algunos de los comandos que existen en bash para darnos una ayuda o en caso de poner un comando seguido de help nos dice que cosas se pueden hacer con ese comando.

rm: nos sirve para poder eliminar un archivo.

touch: este comando sirve para poder crear un archivo.

history: nos muestra todos los comandos que hayamos utilizado en la terminal desde que la abrimos. 

pwd: nos muestra el directorio en el cual estamos trabajando.

git --version: nos muestra la version de git que tenemos instalada.

git init: crea un nuevo repositorio de git en el directorio actual.

git status: Nos muestra los archivos que estan en el staging area, los archivos que se le han hecho modificaciones y que no cuentan con un stalker.

git add: agrega el archivo que nosotros seleccionemos para pasarlo al staging area y despues proceder a hacer un commit.

git commit: 

git restore: restaura un archivo a una version anterior.

git branch: sirve para crear una nueva rama.

git checkout: sirve para movernos entre las ramas

git global --global user.name/user.email: sirve para configurar la cuenta que va a estar trabajando en git.

git log --oneline: muestra en una sola linea los commits que se han hecho.

git merge: sirve para fusionar ramas

clear: limpia la consola.

code . : abre visual studio code.


~~~
